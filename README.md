# Google Calendar Clone

This project is a full-stack, responsive clone of Google Calendar. It allows users to view, create, edit, and delete events in a modern calendar interface. The application uses React and TypeScript on the frontend, with Supabase providing the backend database and real-time capabilities.

![Screenshot of the Google Calendar Clone application]
![alt text](<Screenshot (460).png>)

## Features

* **View Switching:** Dynamically switch between Monthly, Weekly, and Daily calendar views.
* **Event CRUD:** Full Create, Read, Update, and Delete functionality for calendar events.
* **Persistent Storage:** All events are saved to a Supabase (PostgreSQL) backend.
* **Real-time Updates:** (If Supabase is configured) Events update in real-time.
* **Sidebar Mini-Calendar:** A quick navigation-sized calendar, just like the real Google Calendar.
* **Responsive Design:** Built with Tailwind CSS, the UI is clean and usable on various screen sizes.

## Tech Stack

* **Frontend:** React, TypeScript, Vite
* **Backend:** Supabase (PostgreSQL, Real-time)
* **Styling:** Tailwind CSS
* **UI Components:** shadcn-ui

### Prerequisites

* Node.js (v18 or higher)
* npm (or yarn/bun)
* A free [Supabase](https://supabase.com/) account

---

### Installation & Setup

**1. Clone the Repository**
```bash
git clone [https://github.com/Prakhar-998/Google_Calender_clone.git](https://github.com/Prakhar-998/Google_Calender_clone.git)
cd Google_Calender_clone
```

**1. Dependencies**
```bash
npm install
```

**3.Run**
```bash
npm run dev
```
